📌 Overview
This repository contains my end-to-end practice with NumPy, covering beginner to challenge-level concepts. It includes exercises on:
- Array creation
- Indexing and slicing
- Boolean and fancy indexing
- Broadcasting
- Challenge problems combining multiple techniques
The goal of this project is to strengthen my understanding of NumPy for data analysis and scientific computing.

📂 Contents
- numpy_practice.py → All exercises solved step by step
- Organized by levels:
- Beginner
- Intermediate

🧑‍💻 Concepts Covered
- Array Creation: np.arange, np.linspace, np.ones, np.identity
- Indexing & Slicing: row/column extraction, reversing arrays
- Advanced Indexing: fancy indexing, boolean masks
- Broadcasting: applying operations across arrays
- Challenge Problems: combining slicing, masking, and reshaping

- Practiced tricky areas like boolean indexing and axis rules
- Prepared a strong foundation for moving into pandas and deeper data analysis
